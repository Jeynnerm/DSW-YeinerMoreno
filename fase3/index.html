Propósito del Sitio


El propósito del sitio es elaborar e implementar recursos educativos digitales para dinamizar las estrategias de aprendizaje, apoyándose en la construcción de estos recurso con el tema de Etiquetas semánticas, CSS Grid.


Introducción


La tecnología de diseño web mediante CSS nos permite controlar la posición de los elementos contenidos en la página web en relación a sus posiciones predeterminadas en el proceso de diseño normal, así como el resto de elementos a su alrededor, su contenedor principal o vista / ventana principal. Cada tecnología tiene sus usos, ventajas y desventajas, y ninguna tecnología está diseñada para usarse sola. Si comprende el propósito del diseño de cada método, comprenderá fácilmente cuál es la mejor herramienta de diseño para cada tarea.
